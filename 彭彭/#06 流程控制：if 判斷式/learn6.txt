流程控制:判断式

# if 判断式

基本语法1
if 布林值:
    若布林值为 True, 执行命令 
#只执行 True
#前面的空白称之为"缩排" 可用 Tab 来做 "缩排" 这个动作 
# "缩排"非常的重要 它代表跟判断式的概念连接在一起 当布林值是 True, 执行命令  

基本语法2
if 布林值:
    若布林值为 True, 执行命令 
else:
    若布林值为 False, 执行命令

# 执行 True 和 False
# 当布林值是 False, 执行命令 

基本语法3
if 布林值1:
    若布林值1为 True, 执行命令 #满足第1个条件跑第1段
elif布林值2:
    若布林值2为 True, 执行命令 #第1个条件不满足 就来看第2个条件 #如果满足第2个条件 跑第2段
else:
    若布林值1和2都 False, 执行命令 #如果第1个和第2个条件 都是False 都不满足 才跑第3段
# if elif else
#可以做比较复杂的动作,条件,判断式

程式范例
x=input("请输入数字:") #基本输入为字串形态
x=int(x) #转换为整数形态
if x>200:
    print("大于 200")
elif x>100:
    print("大于 200, 小于 200")
else：
    print("小于 100")
